# 🚀 Microservices DevOps Platform

Production-grade cloud-native DevOps platform demonstrating containerization, CI/CD, cloud deployment, Kubernetes orchestration, monitoring, and GitOps practices.

---

# 📌 Project Overview

This project simulates a real-world DevOps infrastructure used in modern cloud-native applications.

The platform includes:
- Frontend application
- Backend API
- Docker containerization
- CI/CD pipelines
- AWS cloud deployment
- Kubernetes orchestration
- Infrastructure as Code
- Monitoring & Logging
